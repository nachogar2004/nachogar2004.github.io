# nachogar2004.github.io

## Practica 4 (PAT)

Sitio web de pedidos de cuadros de arte implementado con HTML + CSS.

Estructura creada en `p4/`:

- `p4/index.html` (pagina principal)
- `p4/catalogo.html` (tabla de cuadros)
- `p4/multimedia.html` (contenido multimedia)
- `p4/pedido.html` (formulario web)
- `p4/css/styles.css` (estilo comun)

## Publicacion en GitHub Pages

1. Haz push de estos cambios al repositorio `nachogar2004.github.io`.
2. En GitHub, abre `Settings > Pages`.
3. En `Build and deployment`, selecciona:
	- Source: `Deploy from a branch`
	- Branch: `main` (o `master`) y carpeta `/ (root)`
4. Guarda los cambios y espera a que termine el despliegue.

URL esperada de la practica:

`https://nachogar2004.github.io/p4/`
