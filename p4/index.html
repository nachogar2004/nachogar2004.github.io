Pedidos de cuadros de arte personalizados

Bienvenido a la practica 4 de PAT. Esta web simula una galeria online donde puedes consultar cuadros disponibles, ver contenido visual de las obras y realizar un pedido personalizado.

El objetivo es combinar una navegacion clara con un estilo uniforme, integrando HTML y CSS para presentar una experiencia de compra artistica.

Ver catalogo de cuadros

Resumen de la galeria

Galeria Atelier reune obras originales en distintos formatos: oleo, acuarela y tecnica mixta. Cada cuadro incluye informacion de tamano, precio y disponibilidad para facilitar tu pedido.

  • Cuadros listos para envio y opcion de encargo personalizado.
  • Asesoramiento para elegir estilo y paleta de color.
  • Entrega a domicilio con seguimiento del pedido.
